## Temas abordados:

#### HTML5:
- container para outras tags - head;
- título - title;
- cabeçalhos (headers);
- sessões (section);
- fonte de título (h1, h2, ...);
- corpo (body);
- tag de navegação (nav);
- marcadores (ul);
- lista de itens (li);
- paragrafo (p);
- hyperlink (a);
- divisão (div);

#### CSS3:
- tamanho da fonte (font-size);
- cor de fundo (background);
- margens (margin.*);
- cores (color);
- tamanho (width);
- Aplicando CSS em classes (.) e elementos (#).
